

# Topic
y'' + y = x^2

---

### 解題步驟

#### 1️⃣ 齊次解
y'' + y = 0

特徵方程：
r^2 + 1 = 0  
⇒ r = ±i

⇒ y_h = C1 cos(x) + C2 sin(x)

---

#### 2️⃣ particular 解（待定係數法）
猜：
y_p = Ax^2 + Bx + C

---

#### 3️⃣ 計算導數
y_p' = 2Ax + B  
y_p'' = 2A  

---

#### 4️⃣ 代回原式
y'' + y = x^2

⇒ 2A + (Ax^2 + Bx + C) = x^2

⇒ Ax^2 + Bx + (C + 2A) = x^2

---

#### 5️⃣ 比較係數
A = 1  
B = 0  
C + 2A = 0 ⇒ C = -2  

---

#### 6️⃣ particular 解
y_p = x^2 - 2

---

### ✅ 最終解
y = C1 cos(x) + C2 sin(x) + x^2 - 2

---
