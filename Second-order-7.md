

# Topic

y'' + y = x^2

---


y'' + y = 0

## feature

r^2 + 1 = 0  
⇒ r = ±i

⇒ y_h = C1 cos(x) + C2 sin(x)

---

## particular 
猜：
y_p = Ax^2 + Bx + C

---

y_p' = 2Ax + B  
y_p'' = 2A  

---

## return 
y'' + y = x^2

⇒ 2A + (Ax^2 + Bx + C) = x^2

⇒ Ax^2 + Bx + (C + 2A) = x^2

---

A = 1  
B = 0  
C + 2A = 0 ⇒ C = -2  

---

## particular 
y_p = x^2 - 2

---
